# Hello World Example

A simple Visual Studio Code extension that displays a Hello World notification.

## Features

Open the Command Palette and run:

`Hello World: Say Hello`

A notification will appear with a Hello World message.

## Usage

1. Press `Ctrl + Shift + P`.
2. Search for `Hello World: Say Hello`.
3. Press Enter.
